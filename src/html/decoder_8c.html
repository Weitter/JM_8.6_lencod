Finds the reference MB given the decoded reference frame.

Note
This is based on the function UnifiedOneForthPix, only it is modified to be used at the "many decoders in the encoder" RD optimization. In this case we dont want to keep full upsampled reference frames for all decoders, so we just upsample when it is necessary.
Parameters
imYThe frame to be upsampled
block_yThe row of the block, whose prediction we want to find
block_xThe column of the block, whose prediction we want to track
mvhorMotion vector, horizontal part
mvverMotion vector, vertical part
outOutput: The prediction for the block (block_y, block_x)
